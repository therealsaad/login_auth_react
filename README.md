# login_auth_react
A simple authentication system built with React, MongoDB, and JWT featuring user registration and login functionality.
This repository contains a basic authentication system built with React and MongoDB. It implements user registration and login functionality using JSON Web Tokens (JWT) for authentication. The project focuses on understanding secure authentication flows, backend integration, and token-based authorization.
